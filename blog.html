Question-1: What are the different ways to select an element in the DOM?

document.getElementById("id") → Selects an element by its ID.
document.getElementsByClassName("class") → Selects elements by class name.
document.getElementsByTagName("tag") → Selects elements by tag name.
document.querySelector("selector") → Selects the first matching element.
document.querySelectorAll("selector") → Selects all matching elements.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML → Retrieves or sets the HTML content inside an element (includes tags).
innerText → Retrieves or sets only the visible text inside an element (ignores hidden text).
textContent → Retrieves or sets all text content inside an element (including hidden text).

Question-3: What is event delegation in the DOM?

A technique where a parent element handles events for its child elements using event bubbling.
Useful for dynamically added elements.
Implemented by adding an event listener to a common ancestor and using event.target to detect the clicked child.

Question-4: What is event bubbling in the DOM?

When an event triggered on a child element propagates up through its ancestors.
Example: Clicking a button inside a div triggers the click event on both the button and the div.
Can be stopped using event.stopPropagation().

Question-5: How do you create, add, and remove elements using JavaScript?

Creating an element → document.createElement("tag").
Adding an element → parent.appendChild(child) or parent.append(child).
Removing an element → element.remove() or parent.removeChild(child).
